Introduction to C

What is C?

C is a procedural, general-purpose computer programming language developed in 1972 by Dennis Ritchie.

“C is quirky, flawed, and an enormous success.” – Dennis Ritchie

It was created to write the UNIX operating system but has since become one of the most widely used programming languages of all time.

Why C Was Created?

Dennis Ritchie created C to:

  • Build utilities running on Unix
  • Provide low-level access to memory
  • Have a simple and consistent language structure
  • Compile to efficient machine instructions
The Mother Language: Many modern languages like C++, C#, Java, and Python borrow fundamentally from C's syntax and concepts.

Language Characteristics

Feature Meaning
Middle-level Bridges the gap between machine-level execution and human-readable code.
Compiled Transforms fully into machine readable executables for fast execution.
Statically typed All variables must be explicitly defined with a data type before usage.
Procedural Follows a top-down execution approach relying on functions and actions.
Portable C code can run on a multitude of architectures with minimal changes.
Fast and Efficient Direct access to system-level operations ensures high-speed execution.

Execution & Memory Management

How C Code Runs

  • Source code (.c) is written
  • The Compiler (e.g. GCC) translates it to an object file (.o)
  • The Linker links object files and produces an executable (.exe)
Performance Note: This direct compilation to machine language makes C extremely fast, but it means you must manage memory carefully.

Pointers and Memory

C provides direct access to system memory through pointers.

c
int x = 10;
int *p = &x;
// 'p' holds the exact memory address of the variable 'x'
// '*p' will access the value stored at that address

Core Data Types

Primary Types

int, float, char, double

Fundamental data types that support individual values in memory. 

int age = 20;
float pi = 3.14f;
char myGrade = 'A';

Derived Types

array, pointer, structure, union

Complex data structures that group or point to primary data types.

int marks[3] = {90, 85, 95};
struct Point { int x; int y; };

Control Flow & Loops

Conditional Statements

Control flow structures are used to handle logical pathways inside functions.

c
int age = 18;

if (age >= 18) {
    printf("You are an adult\n");
} else if (age > 12) {
    printf("You are a teenager\n");
} else {
    printf("You are a child\n");
}
Semicolons: C heavily relies on semicolons (;) to properly terminate statements, completely regardless of whitespace or indentation.

Looping Constructions

For Loop

Used when you know exactly how many iterations to complete.

for (int i = 0; i < 5; i++) {
    printf("%d\n", i);
}

While Loop

Evaluate looping iterations conditionally on an ongoing truth.

while (health > 0) {
    playGame();
}
