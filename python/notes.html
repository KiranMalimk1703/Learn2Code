Introduction to Python

What is Python?

Python is a high-level, interpreted, general-purpose programming language designed with the philosophy:

“Code should be readable and simple.”

Unlike low-level languages such as C and Assembly, Python hides hardware complexity and allows the programmer to focus entirely on logic and problem-solving.

Why Python Was Created?

Guido van Rossum created Python to:

  • Reduce code complexity
  • Improve readability
  • Allow rapid development
  • Make programming accessible to beginners
The Zen of Python: Simple is better than complex. Readability counts. Explicit is better than implicit.

Language Characteristics

Feature Meaning
High-level No memory management required by the user
Interpreted Executes line by line for easier debugging
Dynamically typed No need to declare data types explicitly
Object-oriented Everything in Python is strictly an object
Portable Write once, runs on any operating system
Extensible Can easily integrate with C/C++ functionality

Execution & Memory Management

How Python Code Runs

  • Source code (.py) is written
  • Python interpreter converts it into bytecode
  • Bytecode runs on the Python Virtual Machine (PVM)
Performance Note: This is why Python is slower than C, but much faster to develop in terms of human hours.

Variables & Objects

A variable is a reference to a memory object, not a physical container.

python
x = 10
# 10 is an integer object in memory
# 'x' points to that exact object location

Core Data Types

Immutable Types

int, float, str, tuple

Cannot be physically changed after creation. Any "change" creates a brand new memory object. 

x = 5
x = x + 1  # new object created!

Mutable Types

list, set, dict

Can be extensively modified in place without creating new system objects.

lst = [1, 2]
lst.append(3) # modified in place

Control Flow & Loops

Conditional Statements

Conditional statements are used to control program behavior dynamically based on certain logical conditions.

python
age = 18

if age >= 18:
    print("You are an adult")
elif age > 12:
    print("You are a teenager")
else:
    print("You are a child")
Indentation: Python uses indentation instead of curly braces {}, which elegantly enforces clean and readable logic.

Looping Constructions

For Loop

Used when the exact number of iterations is known.

for i in range(5):
    print(i) # prints 0 to 4

While Loop

Used when iterations heavily depend on an ongoing condition.

while battery > 0:
    run_program()
