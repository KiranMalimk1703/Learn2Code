Introduction to C++

What is C++?

C++ is a cross-platform, powerful, general-purpose programming language created by Bjarne Stroustrup as an extension of the C programming language.

“C makes it easy to shoot yourself in the foot; C++ makes it harder, but when you do it blows your whole leg off.” – Bjarne Stroustrup

It provides developers with a high level of control over system resources and memory, making it the choice for high-performance applications like video games or operating systems.

Why C++ Over C?

  • Supports Object-Oriented Programming (OOP)
  • Much richer standard library (STL)
  • Exception Handling for errors
  • Function and Operator overloading
Compatibility: Almost all C programs are valid C++ programs. C++ is essentially a superset of C!

Key Additions & I/O

Basic Input and Output

C++ introduces iostream, moving away from C's printf() and scanf().

cpp
#include <iostream>
using namespace std;

int main() {
    int age;
    cout << "Enter your age: "; // Printing to console
    cin >> age;                 // Reading from user
    
    cout << "You are " << age << " years old!" << endl;
    return 0;
}

References vs. Pointers

C++ introduces References, which are safer and cleaner alternatives to pointers.

int x = 10;
int& ref = x; // ref is an alias to x
ref = 20;     // x is now 20, no dereferencing needed!

Object-Oriented Principles

The Core Pillars of OOP

Encapsulation

Bundling data and methods that work on that data within a single class unit, hiding the internal state from the outside world using access modifiers (public, private).

Inheritance

Allows one class (derived) to inherit attributes and methods from another class (base). This naturally supports code reusability.

Polymorphism

The ability to present the same interface for differing underlying forms (data types or logic). Often achieved via virtual functions.

cpp
class Animal {
public:
    virtual void makeSound() {
        cout << "Some sound" << endl;
    }
};

class Dog : public Animal {
public:
    void makeSound() override {
        cout << "Bark!" << endl;
    }
};

The STL (Standard Template Library)

The STL is a massive library of generic classes and functions built deeply using C++ templates. It heavily reduces the pain of writing data structures from scratch.

Vectors

Dynamic arrays that resize automatically.

vector<int> nums = {1, 2, 3};
nums.push_back(4); // Dynamically grows

Maps

Collections of Key-Value pairs (Dictionaries).

map<string, int> ages;
ages["John"] = 25;
